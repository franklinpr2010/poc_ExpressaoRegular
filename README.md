## Metacaracteres

Ponto (.) = Um caractere qualquer  
[] - Conjunto de caracteres.  
[^] - Conjunto negado de caracteres  
? - Opcional  
'*' - Zero ou mais.  
'+' - Mais, um ou mais.  
{n, m} - De n até m.   
^ - Inicío da linha.  
$ - Fim da linha.  
\b - Inicio ou fim da palavra  
\ - Uso de metacaracteres como literal.  
| - Operação de Ou  
(  ) - Define um grupo 
\1...\9 - Resgata grupos já definidos
